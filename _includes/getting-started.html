Getting started

First, log in and apply for a key. Your request should be reviewed within 48 business hours. For now, each project is allotted 10,000 API requests per day. All of your requests to the API must include your key using the api_key parameter.

The base URL for the Course Data API is:

http://api.asg.northwestern.edu/

To retrieve data from the API, send an HTTP GET request to the base URL + an endpoint. For instance, pasting http://api.asg.northwestern.edu/terms/?api_key=YOUR_KEY in your browser will return a JSON string representing a list of terms. Making the same request would look like this using JavaScript with jQuery:

{% highlight javascript %} var params = { api_key: YOUR_KEY }; $.getJSON('http://api.asg.northwestern.edu/terms/', params, function(data) { // Prints response to the console console.log(data); }); {% endhighlight %}

Using Python with the requests library:

{% highlight python %} import requests params = { 'api_key': YOUR_KEY } response = requests.get('http://api.asg.northwestern.edu/terms/', params=params) print response.json() {% endhighlight %}
