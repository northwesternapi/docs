Courses

Returns a list of courses.

Examples

To find all Spanish courses offered during Winter 2014, send a request to /courses/ with the parameters term set to 4530 and subject set to SPANISH, i.e. http://api.asg.northwestern.edu/courses/?api_key=YOUR_KEY&term=4530&subject=SPANISH.

Parameters

Name Description
term required id of the term for which to return courses
subject symbol of a subject
instructor id of an instructor
room id of a room
course_num course number e.g. 101-3 in CHEM 101-3. Accepts one or more characters to use as a filter
meeting_days TODO __including, __excluding?
start_time
end_time
start_date
end_date
seats number of seats in the course
section_num section number of a given course
component e.g. LECTURE, LAB

The start_time, end_time, start_date, and end_date expect exact values by default. You can append __lt or __gt to each of them to filter by values that are less than or greater than the value you specify.

Response format
