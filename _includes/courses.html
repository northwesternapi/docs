Courses

Returns a list of courses.

Examples

To find all Spanish courses offered during Winter 2014, send a request to /courses/ with the parameters term set to 4530 and subject set to SPANISH, i.e. http://api.asg.northwestern.edu/courses/?api_key=YOUR_KEY&term=4530&subject=SPANISH.

Parameters

Name Description
id unique id of the course, used only by this API
term required id of the term for which to return courses
subject symbol of a subject
instructor id of an instructor
room id of a room
catalog_num departmental catalog number of the course, including the sequence number
meeting_days TODO __including, __excluding?
start_time
end_time
start_date
end_date
seats number of seats in the course
component e.g. LECTURE, LAB
section a particular occurrence of this course. Used to help identify the course if multiple professors or times are offered in a given term
class_num University's class number for the course, not unique
course_id University's id number for the course, not unique

The start_time, end_time, start_date, and end_date expect exact values by default. You can append __lt or __gt to each of them to filter by values that are less than or greater than the value you specify.

Response format

Course data is returned in lists of objects. Each object has the following structure:

{% highlight json %} { "id": 78818, "title": "Music and Mind", "instructor": "Richard D Ashley", "subject": "MUS_THRY", "catalog_num": "251-0", "section": "20", "room": "Music Administration Bldg 229", "meeting_days": "MoWeFr", "start_time": "09:00:00", "end_time": "09:50:00", "seats": 20, "topic": null, "component": "LEC", "class_num": 16834, "course_id": 19151 } {% endhighlight %}
