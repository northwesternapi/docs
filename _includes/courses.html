Courses

Returns a list of courses.

Examples

To find all Spanish courses offered during Winter 2014, send a request to /courses/ with the parameters term set to 4530 and subject set to SPANISH, i.e.

http://api.asg.northwestern.edu/courses/?api_key=YOUR_KEY&term=4530&subject=SPANISH

Parameters

To query the /courses/ endpoint, you must include a minimum set of parameters. These combinations are listed below. You can include additional parameters to narrow down your search, but the set of parameters must include one of these combinations.

  • instructor
  • id (there can be multiple)
  • term, subject
  • term, room

Here are descriptions of all available parameters:

Name Description
id id of the course. Uniquely idenfities a course; used only by this API
term id of the term
subject symbol representing the subject of the course
Examples: BIOL_SCI, PHIL
instructor id of the instructor who teaches the course
room id of the room in which the course is held
catalog_num departmental catalog number of the course, including the sequence number
meeting_days days of the week when the course meets
start_time time of day when the course starts
end_time time of day when the course stops
start_date date when the course starts
end_date date when the course stops
seats number of seats available
component what part of the course it is
Examples: LEC, LAB
section a particular occurrence of this course. Used to help identify the course if multiple professors or times are offered in a given term
class_num the University's class number for the course (not unique)
course_id the University's id number for the course (not unique)

The start_time, end_time, start_date, and end_date expect exact values by default. You can append __lt or __gt to each of them to filter by values that are less than or greater than the value you specify.

Response format

Course data is returned in lists of objects. Each object has the following structure:

{% highlight json %} { "id": 78818, "title": "Music and Mind", "instructor": "Richard D Ashley", "subject": "MUS_THRY", "catalog_num": "251-0", "section": "20", "room": "Music Administration Bldg 229", "meeting_days": "MoWeFr", "start_time": "09:00", "end_time": "09:50", "seats": 20, "topic": null, "component": "LEC", "class_num": 16834, "course_id": 19151 } {% endhighlight %}
